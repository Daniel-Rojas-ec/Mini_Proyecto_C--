# Proyecto en C++ 

## Minisistema de control para productos
El siguiente sisitema permite almacenar hasta 10 productos con su precio correspondiente y guardarlos, posteriormente el usuario podra seleccionar una de las 6 opciones propuestas luego de registrar los datos en el programa para asi poder ejecutar la opcion deseada por el usuario hasta que decida salir del programa.

## Funcionalidades principales
- Registro de productos (con sus precios).
- Mostrar productos ingresados.
- Mostrar suma total.
- Mostrar el promedio general de precios.
- Buscar producto.
- Mostrar factura.
- Salir del programa.

## Compilacion o ejecucion del programa
- Llamamos a la carpeta o ya puede estar abierta en la terminal
- Colocamos: g++ MiniProyecto.cpp -o Proyecto
- Luego colocamos: ./proyecto
- Y el programa correria sin ningun problema
